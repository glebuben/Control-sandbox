# CartPole - Lyapunov-based Control

Проект по разработке системы управления перевёрнутым маятником (cartpole) на основе функций Ляпунова.

## 📋 Описание

Система состоит из тележки и маятника, закреплённого на ней. Задача управления:
1. **Swing-up**: Поднять маятник из нижнего положения в верхнее
2. **Stabilization**: Стабилизировать маятник в верхнем неустойчивом положении
